# Tienda online de productos tecnológicos

## Descripción del proyecto.
Proyecto de creación de una tienda online de productos electrónicos.
## Público objetivo.
1. Personas con alta renta.
2. Personas con suficiente capital para un capricho.
3. Personas primerizas en el uso de internet.
## Funcionalidades principales.
- Aplicación web intuitiva y atarctiva.
- *MarketPlace.*
- Uso de economía circular.
## Tecnologías previstas
### Frontend
- HTML y CSS
- JavaScript
### Backend
- MariaDB (provisional)
- PHP
